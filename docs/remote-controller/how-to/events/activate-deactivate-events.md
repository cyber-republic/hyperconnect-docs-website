# Activate or Deactivate Events

#### Please follow the steps described in 'Display All Events' first.

![Screenshot](../images/events-activate-deactivate-event-1.png)

#### To activate an event, toggle the button on the right bottom corner.

![Screenshot](../images/events-activate-deactivate-event-2.png)

#### The event is now active.

![Screenshot](../images/events-activate-deactivate-event-3.png)

#### To deactivate an event, toggle the button on the right bottom corner.

![Screenshot](../images/events-activate-deactivate-event-4.png)

#### The event has now been deactivated.

![Screenshot](../images/events-activate-deactivate-event-5.png)
