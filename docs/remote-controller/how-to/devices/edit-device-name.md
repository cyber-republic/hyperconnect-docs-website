# Edit Device Name

#### Click the 'Devices' menu item on the bottom in the middle.

![Screenshot](../images/devices-edit-device-name-1.png)

#### Click the 'More Options' button for the relevant device.

![Screenshot](../images/devices-edit-device-name-2.png)

#### Then, click 'Edit'.

![Screenshot](../images/devices-edit-device-name-3.png)

#### The current name will be shown.

![Screenshot](../images/devices-edit-device-name-4.png)

#### Edit the name as necessary.

![Screenshot](../images/devices-edit-device-name-5.png)

#### Click 'Update' button.

![Screenshot](../images/devices-edit-device-name-6.png)

#### The device name has now been changed.
